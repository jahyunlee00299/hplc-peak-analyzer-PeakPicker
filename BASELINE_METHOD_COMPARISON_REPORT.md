# 베이스라인 방법 비교 분석 리포트

## 분석 개요

**분석 대상**: 50개 HPLC 크로마토그램 파일
**비교 방법**: 4가지 베이스라인 보정 방법
**분석 일자**: 2025-11-09

### 비교 대상 방법

1. **Hybrid (기존)** - `HybridBaselineCorrector`
   - Valley 검출 + Local Minimum 조합
   - 3가지 방법 중 자동 선택

2. **Adaptive Spline (개선)** - `ImprovedBaselineCorrector`
   - Confidence 가중치 + RT 기반 슬로프 완화
   - 적응형 스플라인 피팅

3. **Robust Spline (개선)** - `ImprovedBaselineCorrector`
   - Outlier 자동 제거 (MAD 기반)
   - 강건한 스플라인 피팅

4. **Linear (개선)** - `ImprovedBaselineCorrector`
   - 단순 선형 보간
   - 빠른 처리 속도

## 방법별 특성 비교

### Hybrid (기존 방법)

**장점**:
- 민감도가 높아 작은 피크도 검출 가능
- 복잡한 크로마토그램에서 많은 피크 검출

**단점**:
- 노이즈를 피크로 오검출 가능성 높음
- 베이스라인이 너무 낮게 설정되어 면적 과대평가 경향
- NC (Negative Control) 샘플에서 false positive 많음

**권장 사용 사례**:
- 미량 물질 검출이 중요한 경우
- 모든 가능한 피크를 탐색해야 하는 경우

---

### Adaptive Spline (개선 방법 - 권장)

**장점**:
- 주요 피크에 집중하여 안정적인 결과
- RT 기반 슬로프 완화로 급격한 변화 방지
- NC 샘플에서 false positive 최소화
- 자동 품질 점수 제공

**단점**:
- 매우 작은 피크는 놓칠 수 있음
- 계산 시간이 약간 더 소요됨

**권장 사용 사례**:
- 정량 분석이 중요한 경우
- 재현성 있는 결과가 필요한 경우
- 대부분의 일반적인 HPLC 분석

---

### Robust Spline (개선 방법)

**장점**:
- Outlier에 강건함
- 노이즈가 많은 데이터에 효과적
- Adaptive와 유사한 피크 수

**단점**:
- 경우에 따라 진짜 피크도 제외할 수 있음

**권장 사용 사례**:
- 노이즈가 많은 샘플
- 베이스라인이 불안정한 경우

---

### Linear (개선 방법)

**장점**:
- 가장 빠른 처리 속도
- 단순하고 예측 가능한 결과
- Adaptive와 유사한 피크 수

**단점**:
- 복잡한 베이스라인에는 부적합
- 급격한 기울기 변화 처리 어려움

**권장 사용 사례**:
- 빠른 스크리닝
- 베이스라인이 단순한 샘플
- 배치 처리 시 속도가 중요한 경우

## 권장사항

### 1. 기본 설정
**Adaptive Spline을 기본 방법으로 사용 권장**

이유:
- 가장 균형잡힌 성능
- NC 샘플에서 false positive 최소화
- 주요 피크 안정적 검출
- 자동 품질 평가 제공

### 2. 샘플 타입별 전략

**Negative Control (NC) 샘플**:
- ✅ Adaptive 또는 Robust Spline 권장
- ❌ Hybrid는 false positive 많음

**복잡한 크로마토그램 (FLOXU 등)**:
- ✅ Adaptive Spline 권장
- 필요시 Robust Spline으로 검증

**단순한 크로마토그램 (대부분 RIBA)**:
- ✅ Linear 또는 Adaptive 모두 적합
- 빠른 처리 필요 시 Linear 선택

**미량 물질 검출 중요**:
- Hybrid 사용 고려
- 단, 수동 검증 필수

### 3. 품질 관리

**다음 경우 결과 재확인 필요**:
- 피크 수가 예상과 크게 다른 경우
- 총 면적 차이가 50% 이상인 경우
- NC 샘플에서 피크가 2개 이상 검출된 경우

**검증 방법**:
1. 여러 방법 결과 비교
2. 시각화로 베이스라인 확인
3. 피크 높이/면적 합리성 판단
